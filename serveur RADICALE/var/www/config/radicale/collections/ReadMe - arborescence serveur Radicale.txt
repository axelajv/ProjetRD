Précision pour le contenu du dossier : var/www/.config/radicale/collections

Il est nécessaire de créer un dossier et d’ouvrir les droits d’écriture suffisant pour chacun des types de calendriers à extraire : 
	• /Enseignants/ 
	• /Etudiants/ 
	• /Filieres/ 
	• /Salles/
En effet, une requête HTTP PUT ne permet pas la création de l’arborescence.
